# react-native-shazam-kit

ShazamKit (iOS) for React Native.

## Installation

```sh
npm install react-native-shazam-kit
```

## Usage

```js
import { multiply } from "react-native-shazam-kit";

// ...

const result = await multiply(3, 7);
```
